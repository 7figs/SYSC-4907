## A Machine Learning–Based Video Player for Streaming Applications

---

## Project Overview
Video streaming services are widely used, providing users with personalized
recommendations based on their watch behaviour, and preferences. These platforms
collect large amounts of user information which raises concerns related to user privacy,
and how their data is being used or stored. The goal of this project is to develop a
web-based video streaming platform that provides personalized recommendations while
keeping user data private. The system uses a machine learning approach, specifically a
content-based filtering method consisting of a user-unique decision tree classifier with a
user-specific scoring mechanism. Initial training data is collected through a cold start
preference survey, after which user-unique movie recommendations are generated using
metadata features such as genres, actors, release year, keywords, and related attributes.
As a user continues interacting with the platform, the model is updated allowing for more
personalized and niche recommendations. To enforce privacy, all user data is stored
locally in the browser instead of on external servers. Our platform combines a Web
Graphical User Interface (GUI), Browser Session Storage, a Flask-based Web Server, a
movie database, a user-unique recommendation system, and a video streaming
component. Overall, the system shows that personalized recommendations can be
provided while maintaining user privacy
